# Bem-vindo(a) à Wiki do Mechanic Craft Calculator!

Esta documentação foi criada para ajudar você a manter e atualizar o projeto da Calculadora de Craft, mesmo se não tiver experiência avançada com programação.

## Estrutura do Projeto

Os dados importantes da calculadora estão organizados de forma simples, separados por categorias dentro da pasta `data/`:

*   **`materials.js`**: Contém as matérias-primas básicas (Plástico Refinado, Sucata, Borracha, etc.).
*   **`components.js`**: Peças de "Stage" feitas a partir de materiais ou componentes menores (ECU, Kit de Freio, etc.).
*   **`mechanic_services.js`**: Os serviços prestados na oficina e os materiais que eles exigem.
*   **`products.js`**: Produtos finais como Lockpicks, Garrafas de Nitro e Kits de Reparo.
*   **`partners.js`**: Lista das facções/empresas parceiras e seus respectivos descontos.
